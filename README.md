# Waste Management System

A machine learning-powered web application designed to automate and optimize waste segregation and management. This project leverages Deep Learning to classify different types of waste, helping users and municipalities dispose of garbage efficiently and responsibly.

---

## 🚀 Features
* **Automated Image Classification:** Uses a Deep Learning model to categorize waste (e.g., Organic, Recyclable, Hazardous).
* **Interactive UI:** A lightweight and responsive dashboard built using Streamlit.
* **Real-time Analytics:** Provides insights and guidelines on proper disposal methods based on the classified waste type.

---

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Machine Learning Framework:** TensorFlow / Keras (CNN Architecture)
* **Language:** Python 3.10+
* **Environment Management:** Virtualenv (`venv`)
